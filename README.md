# ALΞΤΗΙ ΑΙ Business Card Generator

A premium, dark-mode business card generator for Alethi AI - The Assurance Platform for Technology Due Diligence.

## Design Specifications

- **Dimensions:** Standard US Business Card (3.5" x 2") rendered at 3x scale (1050px x 600px) for high-definition clarity
- **Color Palette:**
  - Background: Deep Obsidian Black (`#0F172A`)
  - Accent: Assurance Teal (`#2DD4BF`)
  - Primary Text: White (`#FFFFFF`)
  - Secondary Text: Slate Grey (`#94A3B8`)
- **Typography:** Inter (Clean, geometric, sans-serif)

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React (icons)
- Vite (build tool)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Premium "audit-grade" aesthetic inspired by high-frequency trading terminals
- Front card with minimalist branding
- Back cards for all 6 team members
- High-definition rendering suitable for print
- Responsive layout with CSS drop shadows
